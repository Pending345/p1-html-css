Waarom noem je de eerste HTML-pagina; index.html?

index.html heeft de functie als een pagina waar gebruikers automatisch naartoe worden gestuurd.

Met welke HTML versie werken we nu?

We werken met HTML5(HyperText Markup Language 5)

Met welke CSS versie werken we nu?

We weken met CSS3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Elke browser webcodes op zijn eigen manier interpreteert, en dus zien websites er anders uit op verschillende browsers.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is de populairste browser, en gebruikt minimaal 16GB RAM.
